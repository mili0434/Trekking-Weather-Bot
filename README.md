# Trekking-Weather-Bot
 A desktop weather application built with Python for trekkers and outdoor adventurers. It fetches real-time weather data using the Open-Meteo API, features a dark-themed graphical interface, provides smart packing guides, and logs search history using a local SQLite database.
 
### Features
* **Real-Time Weather:** Fetches live temperature and forecasts via the Open-Meteo API.
* **Interactive GUI:** Built with Python `tkinter` with a dark theme and location dropdown/custom input.
* **Smart Packing Guide:** Dynamically suggests essential gear and clothing based on weather conditions.
* **Search History:** Automatically logs search details using a local SQLite database with a pop-up history view.

### Tech Stack
* Python
* Tkinter
* SQLite3
* Open-Meteo API
